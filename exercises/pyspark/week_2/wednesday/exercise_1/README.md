### Exercise 1

**Goal:** Practice selecting, filtering, and column manipulation.

**Tasks:**

1. Load `main_users.csv`.
2. **Filter:** Keep only users where `age` is 21 or older.
3. **Add Column:** Create a column `is_senior` that is "Yes" if `age > 40`, else "No".
4. **Transformation:** Create a column `id_label` which is the string "EMP-" concatenated with the `uid`.
5. **Clean up:** Select only `id_label`, `name`, and `is_senior`. Remove all other columns.
